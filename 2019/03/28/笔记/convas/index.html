canvas

  • canvas 是html5提供的一种新标签，他是一个举行的画布区，可以用javascript在上面绘画，控制器每一个像素
  • canvas可以控制宽高，但是不要用CSS控制宽高，会出现图片拉伸，可以设置行内样式，设置宽高
    • ie9以上的浏览器才支持canvas
    • canvas上下文context是所有的绘制操作api的入口或者集合
    • 它支持2d和3d(webgl);

      绘制路径

    • canvas坐标系： 从左上角0，0开始为原点
    • 设置绘制的起点（moveTo）
      1. 语法：ctx.moveTo(x,y)
      2. 设置上下文绘制路径的起点，相当于移动画笔到某个位置
      3. 参数x, y都是相当于canvas盒子的最左上角
      4. 绘制线段前必须先设置起点，不然绘制无效
    • 绘制直线（lineTo）
      1. ctx.linTo(x,y);
    • 路径开始和闭合
      1. 开始路径：ctx.beginPath();
      2. 闭合路径
        ctx.endPath();
      3. beginPath的核心作用是将不同绘制的形状进行隔离
    • 描边（stroke）
      1. 语法： ctx.stroke();
      2. 根据路径绘制线，路径只是草稿，真正的绘制必须执行stroke
    • canvas绘制的基本步骤
      • 第一步：获得上下文 =>canvasElem.getContext(‘2d’);
      • 第二步：开始路径规划 =>ctx.beginPath()
      • 第三步：移动起始点 =>ctx.moveTo(x, y)
      • 第四步：绘制线(矩形、圆形、图片…) =>ctx.lineTo(x, y)
      • 第五步：闭合路径 =>ctx.closePath();
      • 第六步：绘制描边 =>ctx.stroke()；
    • 填充（fill）
      1. 语法： ctx.fill();
      2. 填充是将闭合路径的内容太南充具体的颜色，默认黑色
      3. 注意，交叉路径的填充问题“非零环绕原则”
    • 快速创建矩形的方法rect()
      1. 语法：ctx.rect(x,y,width,height);
      • rect方法只是规划了路径，并没有填充和描边
    • 快速创建描边举行和填充矩形
      1. ctx.strokeRect(x,y,width,height)
      2. ctx.fillRect(x,y,width,height)
    • 清除矩形)(clearRect)
      1. 语法：ctx.clearRect(x,y,width,height);
      2. 清除某个矩形内的绘制的内容
    • 绘制原型（arc）
      1. arc()方法创建弧/曲线
      2. 语法：ctx.arc(x,y,r,sAngle,eAngle,counterclockwise);
        • x,y圆心的坐标
        • r 半径的长度
        • sAngl绘制开始的角度，圆心角最右边的点是0度顺时针方向弧度增大
        • eAngle结束的角度，注意弧度是Π
        • counterclockwise：是否是逆时针。true是逆时针，false：顺时针
        • 弧度和角度的转换公式： rad = deg*Math.PI/180;
    • 绘制文字
      context.fillText(text,x,y,maxWidth);
      1. font 设置或者返回文本内容的当前字体属性，使用方法与CSS font的使用方法形同
    • textAlign设置或者返回文本内容的当前对齐方式（start,end,center,left,right）
    •  ctx.fillText() 在画布上绘制“被填充的”文本
    • ctx.strokeText() 在画布上绘制文本（无填充）
    • 绘制图片
    • context.drawImage(img,x,y);
      • 参数说明： x,y 绘制图片左上角的坐标， img是绘制图片的dom对象。
