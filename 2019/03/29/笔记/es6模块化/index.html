es6新特性

  • 函数参数默认值（存在0的问题）

  • 0 false undefined null ‘’ 在做逻辑判断时候都会被当做 false

  • 模板字符串: ${xx}es6 默认占位符 配合 使用,可以自动替换作用域中的变量

  • 解构赋值

  • 对象属性的简写

  • 箭头函数

    1. function可以省略不写
    2. 多个形参用逗号隔开
    3. => 如果后面只有一行指令，意思是返回这个结果
    4. 如果不需要返回直接写逻辑
    5. 如果只有一个形参，（）可以省略不写
    6. 如果没有性参，括号必须写

  • 箭头函数的this

    1. 在箭头函数里，this将继承上下文对象
  • 在严格模式（’use strict’;）下，对象的this不能指向window对象

  • promise 为了避免出现回调js使用promise把回调转换为线性代码，更符合人的阅读习惯

    1. resolve 解决
    2. reject 拒绝

  • 异步请求

    1. ajax也实现了promise的封装

  • es6提供了两个声明变量的关键词

    1. let 局部变量 可以实现块状作用域
    2. const 用于声明常量

  • 继承

    1. class 类
    2. constructor 构造函数
    3. extends 继承

  • 自执行函数传参 在某种情况下可能会改变默认的执行环境，可以把执行环境传递进来

  • 放大模式

    1. 不依赖文件的加载顺序
    2. 无风险添加新的方法（甚至不用关心对象是否存在）
  • 模块化规范
    1. commonJS规范（同步）
      node.js
    2. ADM规范（异步）
      国外封装的为了解决异步问题
    3. CMD（异步）
      国内的封装的解决异步问题
    4. ES6 模块化规范（官方）
    • export与import关键词实现模块化
    • 浏览器不支持
    • node环境也不支持
  • cli ： 命令行 通过 npm init 在当前文件夹创建 package.json文件 ，它是对当前项目的描述，保存了当前项目所有依赖文件的信息

  • es6模块化常用语法

  • filter 过滤 item 遍历每一个对象

  • map 遍历修改某一项

  • forEach 遍历数组, 两个参数，第二个为下标

  • es6通过export default 到处模块

  • es6 通过 import add … from .. 导入模块
